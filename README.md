# NeoDeck - UNIHIKER K10 Cyber Terminal

A portable **cyberpunk hacker terminal** built on the **UNIHIKER K10** (ESP32-S3) running MicroPython.

## Features

- **Matrix Digital Rain** — Full-screen cyberpunk animation
- **WiFi Scanner** — Real-time network discovery with signal bars
- **Live Sensor Suite** — Temperature, Humidity, Light, Accelerometer
- **Camera Preview** — Built-in camera live view
- **Flashlight** — RGB control
- **BLE Scanner** — Bluetooth device detection
- **Fake Brute Force** — Classic hacker animation
- **System Info** — Device specifications
- Fully controlled with physical **A** and **B** buttons

## Hardware

- **Board**: UNIHIKER K10 (ESP32-S3)
- **Display**: 2.8" 240x320 Color LCD
- **Sensors**: Temp/Humidity, Light, Accelerometer, Camera, Microphone
- **Connectivity**: WiFi + Bluetooth 5.0
- **Power**: USB-C (can run on battery with PH2.0 connector)

## Installation

1. Flash the latest **MicroPython firmware** for UNIHIKER K10 (see official docs)
2. Copy `main.py` to the root of the board using Thonny or similar
3. Restart the device — it will boot directly into the terminal

## Usage

- **Button A** → Navigate down / Toggle (in some modes)
- **Button B** → Select / Back / Exit
